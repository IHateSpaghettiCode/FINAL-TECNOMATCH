Preguntas Frecuentes (FAQ)

Respuestas a las preguntas más comunes sobre el uso, problemas o características:

  1. ¿Qué es TecnoMatch? TecnoMatch es una plataforma que permite a los usuarios realizar un test de personalidad MBTI y recibir recomendaciones de carreras relacionadas con el área TIC disponibles en Bogotá.

  2. ¿Necesito crear una cuenta para usar TecnoMatch? Sí. Para guardar tus resultados, recibir recomendaciones y acceder a funciones adicionales, es necesario registrarse con un correo electrónico válido.

  3. ¿El test MBTI es obligatorio para ver las carreras? Sí. Las recomendaciones dependen de tu tipo de personalidad, por lo que primero debes completar el test.

  4. ¿Qué hago si no puedo iniciar sesión? Verifica que tu correo y contraseña sean correctos. Si el error continúa, prueba restablecer la contraseña o consulta la sección de Solución de Problemas.

  5. ¿Puedo cambiar mis datos personales? Sí. En tu perfil puedes modificar tu nombre, correo e información asociada a tu cuenta.

  6. ¿Cómo se generan las recomendaciones de carrera? El sistema cruza tu tipo de personalidad MBTI con una base de programas académicos TIC disponibles en Bogotá y muestra las dos opciones más compatibles con tu perfil.

  7. ¿Por qué no me aparece ninguna recomendación? Esto ocurre si no has completado el test o si hubo un error de conexión al cargar los datos de la base de datos.

  8. ¿Es gratis usar TecnoMatch? Sí. Todas las opciones principales como registro, test MBTI y recomendaciones son gratuitas.

  9. ¿Mis datos están seguros? Sí. TecnoMatch utiliza buenas prácticas de seguridad y almacenamiento para proteger tu información.

  10. ¿Puedo ver mis resultados anteriores? Sí. Puedes acceder al historial de resultados desde tu perfil si esta función está habilitada.

  11. ¿Qué pasa si cierro la página durante el test? El progreso no se guarda, por lo que tendrás que iniciar nuevamente el cuestionario.

  12. ¿En qué dispositivos puedo usar TecnoMatch? TecnoMatch funciona en computadoras, tablets y dispositivos móviles con navegador actualizado.

  13. ¿Qué hago si recibo un error inesperado? Intenta recargar la página. Si persiste, revisa tu conexión o consulta la sección Solución de Problemas.

  14. ¿Qué navegadores son recomendados? Google Chrome, Microsoft Edge o Mozilla Firefox en sus versiones más recientes para asegurar compatibilidad.

  15. ¿Dónde puedo contactar soporte? En el pie de la página encontrarás un enlace a “Soporte” donde podrás enviar un mensaje al equipo técnico.
